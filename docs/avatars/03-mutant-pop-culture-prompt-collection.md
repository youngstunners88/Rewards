# Rewards Avatar Prompt Collection

## 60 original playful, mutant, pop-culture-twisted character prompts

For the Rewards classroom system for children ages 6–12.

## Locked design rules

- Pure cartoon / stylised animation. Zero hyper-realism.
- Bright, loud, colourful and instantly readable on a tablet or Chromebook.
- Every design is an original fictional hero; do not reproduce existing characters, celebrities, brands, club crests or copyrighted costumes.
- Use familiar concepts only as loose inspiration: sports, music, gaming, occupations, animals, food, science and internet-meme energy.
- Mix genders, personalities, skin tones, body types and silhouettes.
- Every character must read clearly at an 80×80px thumbnail.
- Use huge readable expressions: grins, surprised eyes, confidence, joy or determined concentration.
- Use a clean flat colour or simple gradient background.
- Head-and-shoulders or full-body composition as specified by each prompt, centred and isolated.
- No text, watermark, logos, real team names, official kits, weapons, blood, horror or adult themes.

## Global style suffix

Append this to every prompt:

> Original fictional classroom reward avatar, playful non-photorealistic cartoon, stylised animation concept art, thick clean outlines, flat cel shading, saturated cheerful colours, exaggerated readable silhouette, huge expressive eyes and clear emotion, kid-friendly, centred composition, clean simple background, square 1:1 image, no text, no watermark, no logo, no celebrity likeness, no existing copyrighted character, no photorealism, no horror, no violence.

## Prompt engineering notes

### Style modifiers

Use one modifier when useful: warm whimsical storybook animation; thick-line action cartoon; retro rubber-hose cartoon; soft inclusive pastel gem aesthetic; multidimensional comic-book animation; exaggerated comedy animation; mixed-media collage cartoon; polished game-avatar animation.

### Always include

Playful, cartoonish, stylised animation, bold outlines, saturated bright colours, exaggerated proportions, expressive eyes, big emotion, dynamic pose, kid-friendly, original design, centred composition, clean background, cel shading or flat colours, readable at 80×80px.

### Always exclude

Photorealism, hyper-realistic textures, uncanny valley, generic stock-photo pose, sterile corporate style, existing copyrighted characters, celebrity likeness, brand logos, real club crests, official sports kits, readable text, watermark, adult themes, violence, blood, realistic weapons, horror, depressing dark tones.
